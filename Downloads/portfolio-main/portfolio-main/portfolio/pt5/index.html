오픈 API 통합포털

금융결제원은 오픈 파이낸스 시대에 맞춰 API를 활용한 개방형 플랫폼을 통해
금융회사 및 핀테크 기업과 함께 개방형 혁신으로 나아갑니다.

오픈 API 현황

  • 서비스중인 API

    36 개

  • API 이용건수

    89억2천만 건

  • 활용된 핀테크 기업

    257 개
  • 패키지API 이용기관은 서비스에 도입하고 싶은 API 패키지를 선택하고 여러 개의 서비스로 나뉘어진 오픈API를 조합하여 사용함으로써 혁신적인 금융서비스를 보다 빠르게 구현할 수 있습니다.
  • 오픈뱅킹 오픈뱅킹은 핀테크기업이 금융서비스를 편리하게 개발할 수 있도록 참가기관의 금융서비스를 표준화된 형태로 제공하는 인프라로 총7개의 서비스 API와 인증/관리 API를 제공하고 있습니다.
  • 어카운트인포 어카운트인포는 금융당국 정책(국민체감형 금융거래서비스)에 의거 일반 국민의 금융편의 제고를 목적으로 도입된 금융기관 공동서비스로서 국민 누구나 자신의 계좌·카드·보험 등 숨은 금융자산을 한 번에 조회하고, 잔고가 소액인 비활동성계좌는 즉시 해지·잔고이전할 수 있습니다.
  • 금융인증 금융인증서비스는 금융인증서를 안전한 금융결제원 클라우드에 발급·보관하여 언제 어디서나 안전하고 간편하게 이용할 수 있는 인증서비스입니다.
  • 오픈지로 오픈지로는 제휴사(ERP기업, 핀테크기업 등)가 지로 이용기관과 고객을 연계하여, 앱이나 SNS 알림을 통해 요금을 ❶편리하고 저렴하게 고지하고, ❷간편하고 빠르게 납부할 수 있도록 하는 언택트 기반의 고지 · 납부 서비스입니다.
  • 온라인투자연계금융업 중앙기록관리업무 온라인투자연계금융업 중앙기록관리업무는 법령에 의해 온라인투자연계금융업자로부터 제공받은 투자 및 차입정보, 이용자 정보를 집중·관리하는 업무입니다.
  • 마이데이터중계 API시스템 직접 구축이 어려운 신용정보제공이용자의 부담완화를 위하여 마이데이터 중계시스템을 구축하여 마이데이터사업자, 정보수신자 및 정보제공자 간 개인신용정보를 중계하는 서비스 입니다
  • 금융결제데이터개방 금융결제원이 보유한 통계정보 및 금융결제 데이터를 금융회사, 핀테크, 연구기관 등이 활용할 수 있도록 비식별 처리하여 제공합니다.
  • 금융 MAP 금융 MAP은 전국의 ATM과 금융회사 지점 위치 및 운영시간, 제공서비스 등 세부정보를 조회할 수 있는 서비스입니다.
