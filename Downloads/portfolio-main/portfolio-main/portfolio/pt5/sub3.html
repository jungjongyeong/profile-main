잔액조회

개요

사용자는 이용기관이 제공하는 서비스를 통해 본인계좌에대한 잔액 및 출금가능금액을 조회할 수 있습니다.
이용기관은 "잔액조회 API"를 이용하여 사용자가 등록한 계좌의 잔액을 조회하여 사용자에게 제공할 수 있습니다.

처리방식

이용기관은 사용자의 계좌등록 및 계좌등록확인 과정에서 획득한 핀테크이용번호와 Access Token 를 사용하여 API를 호출합니다.

요청메시지 URL

잔액조회 API 는 GET 메소드로 호출됩니다.
