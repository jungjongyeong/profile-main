만국 여행사

프로젝트 기획 배경

사람들은 코로나로 인해 못 갔던 여행을 가고싶어 하는 욕구가 있다는것을 파악.

  • 어디로 떠날지 빠르게 확인하고 싶다.
  • 항공 시간을 바로 확인하고 싶다
  • 장바구니에 담고 변심이 생길시 다른곳을 가고싶다.
  • 로그인 하면 장바구니에 담긴 여행지를 확인하고 싶다.
  • 남녀노소 사용하기 쉬운 UX/UI로 누구나 여행지를 다녀오 싶다

프로젝트 상세 소개

💡 핵심 기능

  1. 자신이 다녀온 여행지 리뷰, 댓글이 가능합니다.
  1. 장바구니에 상품 저장이 가능합니다.
  1. 간편하게 여행지 검색이 가능합니다.
  1. 고객센터에 질문하기가 가능합니다.

🙋🏻‍♂️ 타겟 유저

  • 10대~50대 거의 모든 유저 가능

📍 기대 효과

  1. 고객들이 간편하게 여행지를 검색하고 다녀올수 있습니다.
  1. 지금 시즌에 뜨고있는 여행지를 유저가 확인하여 다녀올수 있습니다.

개발 기간 & 일정

  • 개발 기간 : 2023.1.18 ~ 2023.4.15

사용한 기술 스택

프론트엔드

  • Javascript, React
  • Redux-toolkit, Axios
  • FontAwesome
  • ESLint, Prettier, Netlify

백엔드

담당한 파트

FE 전체 개발, BE 전체 개발, 모든페이지 기획, 디자인

프론트엔드

  • 유저관련 페이지(장바구니)
  • 상품 리스트를 보여주는 홈페이지
  • 리뷰를 생성 모달 컴포넌트

백엔드

  • 유저, OAuth 구글 인증
  • 댓글 생성 구현
  • 장바구니 저장 데이터 구현
  • Axios의 활용으로 항공 데이터 추출
  • 상품들 JSON파일로 저장 후 추출
